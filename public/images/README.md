# Images Directory

This EPK project references the following image files that need to be added:

1. `screenshot-2026-02-06-at-24042-am.png` - Hero background and main press photo
2. `03302837-dd6f-40b1-b790-7561fc11d262.png` - Stage performance photo
3. `images.jpeg` - Gallery image
4. `hq720.jpg` - Gallery image

## Usage

Please add your own artist photos to this directory with the names above, or update the image paths in the components:
- `/src/components/Hero.tsx`
- `/src/components/Bio.tsx`
- `/src/components/Gallery.tsx`
